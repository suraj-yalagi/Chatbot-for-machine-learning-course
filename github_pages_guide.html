Host Your ML Chatbot on GitHub Pages

This guide will walk you through hosting your ML Chatbot on GitHub Pages, which is a free and easy way to get your chatbot online with a shareable link.

1 Create a GitHub Account

Go to GitHub's signup page and create a free account if you don't already have one.

2 Create a New Repository

After logging in to GitHub:

  1. Click the "+" icon in the top-right corner and select "New repository"
  2. Name your repository (e.g., "ml-chatbot")
  3. Make it public
  4. Click "Create repository"
3 Upload Your Files

On your new repository page:

  1. Click "uploading an existing file"
  2. Drag and drop your files from the ml_chatbot_hosting folder
  3. Add a commit message like "Add ML chatbot files"
  4. Click "Commit changes"
4 Enable GitHub Pages

After uploading your files:

  1. Go to your repository's "Settings" tab
  2. Scroll down to the "GitHub Pages" section
  3. Under "Source", select "main" branch
  4. Click "Save"
5 Get Your Shareable Link

After a few minutes, your site will be published at:

https://[your-username].github.io/[repository-name]/

This is your shareable link!

6 Share Your Chatbot

Now you can share your chatbot with anyone by sending them the GitHub Pages URL. They can access it with a single click!

Note: GitHub Pages is completely free and includes:
  • Free hosting for static websites
  • Custom domain support (optional)
  • HTTPS/SSL certificates
  • Automatic updates when you push changes to your repository
